Run as a Java Microservice
-----------------------------
To run the application, 
 - update the configuration variables in the application.properties file to align with your setup,
 - Afterward, execute the snt.cmd command.
remark: The files (application.properties,snt.cmd) are located in the root directory of the application.


Requirements:
Java OpenJDK 17 or higher
MySQL 8 or higher

Run Within a Docker Container
------------------------------
To test the application within a Docker container:

-Update the compose-vars.env file to reflect your configuration,
-Run the snt-docker-pull.cmd command.
remark: The files (compose-vars.env,snt-docker-pull.cmd)are located in the root directory of the application.
